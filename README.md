# Homebrew tap for Seriesly

[Seriesly](https://series-ly.com) — private, local meeting memory for Mac.

## Install

```sh
brew tap harrisonju123/seriesly
brew install --cask seriesly
```

Updates are delivered in-app via Sparkle (`auto_updates true`), or with `brew upgrade --cask seriesly`.

This cask is updated automatically on every release by the Seriesly release pipeline.
